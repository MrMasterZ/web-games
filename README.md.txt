Это WEB клиент-серверная игра "Города".
Написана в IDE eclipse.
Используемые в ней технологии и языки: html5, css3, javascript 1.5, jquery-3.3.1, ajax, java 1.8.0, JDBC, PostgreSQL 9.6, Apache Tomcat 9.0
БД городов была загружена из приложенного файла script.sql
При запущенном через eclipse Tomcat открыть игру: http://localhost:8080/gamecities/cities.html
